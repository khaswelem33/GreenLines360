# Media Assets Guide

This project uses physical PNG placeholders for all replaceable page imagery.

To replace an image later:

1. Export the real image as PNG.
2. Save it over the matching file in `public/images/.../`.
3. Keep the same file name.
4. Rebuild or redeploy the site.

No content or code changes are required when the file name stays the same.
